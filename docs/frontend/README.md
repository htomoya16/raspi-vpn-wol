## 運用時の注意点

- モバイル表示（幅 760px 以下）は下部ナビで単一パネル表示、PC表示（幅 761px 以上）は従来どおり2カラム表示になる。
- 1ステップごとに動作確認を行い、問題があれば次へ進む前に修正する。
- API I/F 変更時はフロントとバックエンドを同時更新する。
- 配布対象は `frontend/dist/`。成果物はGit管理しない。
- コマンド運用: 読み取り系（例: `ls` `rg` `cat` `sed` `git status`）はCodex側で実行可。
- コマンド運用: 実行系（例: `npm` `python` `pytest` `uvicorn`）は学習者が実行する。
